# Homelab & Linux Showcase

Hi, I'm Aidan. Over the past two months I've been learning Linux, self-hosting,
and low-power homelabbing across multiple devices and architectures.

This repository documents my setup, decisions, and what I’ve learned.

---

## Hardware Overview

### Raspberry Pi (Primary Homelab)
- OS: Raspberry Pi OS / Debian-based
- Role: Docker host, services, web hosting

### 2014 MacBook Air
- OS: Linux Mint
- Window Manager: i3
- Focus: Linux workflow, tiling WM configuration

### 2013 Surface RT
- OS: postmarketOS
- Architecture: ARM
- Focus: Running Linux on unsupported hardware
